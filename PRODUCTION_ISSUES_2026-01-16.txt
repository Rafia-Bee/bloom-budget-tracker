1. [CRITICAL] WORKER TIMEOUT ON /api/v1/currencies/rates
   --------------------------------------------------------
   - Status: Worker crash (SIGKILL) causing 500 error
   - Location: backend/routes/currency.py, backend/services/currency_service.py
   - Cause: Endpoint loops through 165 currencies, making individual API calls
            Each external API call has 10s timeout, total exceeds gunicorn's 30s limit
   - Error: "WORKER TIMEOUT (pid:39)" followed by SystemExit: 1
   - Impact: Currency rates fail to load, CORS error shown to user (500 response)
   - Fix: Refactor to use refresh_rates() which fetches ALL rates in single API call

2. [CRITICAL] CORS ERROR ON CURRENCY RATES FAILURE
   ------------------------------------------------
   - Status: CORS policy blocking response
   - Error: "Access-Control-Allow-Origin header not present on 500 response"
   - Cause: When worker crashes, error response doesn't include CORS headers
   - Impact: Users see currency fetch failure after login
   - Fix: Ensure error responses include CORS headers OR fix root cause (#1)

3. [HIGH] PWA ICON ERROR - ROOT CAUSE FOUND
   ------------------------------------------

   - Location: frontend/public/icon-192.png, frontend/public/icon-512.png

   - Note: bloomLogo2.png (262KB) can be used as source image
